# 3DLOOK — Color Palette

> Extracted from FitXpress deck and 3DLOOK brand presence. Vadim please confirm exact HEX values from official brand book if available.

## Primary

- **3DLOOK Blue:** `#2962FF` (approximate from deck — confirm with Vadim/Figma)
  - Used for primary CTAs, accent UI elements, FitXpress branding
- **3DLOOK Black/Near-black:** `#0A0A0A` to `#000000`
  - Background of hero / dark sections
  - Headings on light backgrounds
- **White:** `#FFFFFF`
  - Backgrounds, light sections, contrast for blue/black

## Secondary

- **Light Gray:** `#F5F5F7`
  - Section backgrounds, subtle separation
- **Mid Gray:** `#6B7280` (approximate)
  - Secondary text, labels

## Accent / data viz (sparingly)

- **Red Alert:** `#FF3B30` (approximate)
  - "Mismatch" badges, warning states (Smart Scales mismatch)
- **Green Success:** `#34C759` (approximate)
  - Verified states, positive deltas

## Usage rules

### Do
- Lead with **3DLOOK Blue + Black + White** for hero / promotional content
- Use blue for CTAs, headlines, key data points
- Use ample white space — dark space when going premium

### Don't
- **NO purple-pink gradients** (typical AI-generic aesthetic — explicitly banned)
- **NO multi-color rainbows** in data viz — stick to brand colors + 1 accent max
- **NO low-contrast text** (always ≥ 4.5:1 contrast ratio)
- **NO Comic Sans / decorative fonts** (see fonts.md)
